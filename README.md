# 개발자 전용 쇼핑몰 「dev store」

```
💻 데브 스토어는 개발자 용품을 파는 온라인 쇼핑몰입니다.

🚀 바닐라 자바스크립트 SPA 프로젝트입니다. 아직 진행중이에요!
```

#### [기술]

-   FrontEnd: vanilla JavaScript, Sass
-   BackEnd: 제공된 API 사용
-   vite

#### [개발 환경]

-   vscode
